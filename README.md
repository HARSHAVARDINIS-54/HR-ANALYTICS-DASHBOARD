
# HR Analytics Dashboard – Power BI Project

## Project Overview

This project is an HR Analytics Dashboard developed using Microsoft Power BI to analyze employee attrition and workforce trends. The dashboard is built using an employee dataset and provides insights into factors influencing employee turnover, job satisfaction, and overall workforce distribution.

The objective of this project is to apply data analytics techniques to human resource data and support data-driven decision-making in employee management.

---

## Tools and Technologies Used

* Microsoft Power BI (.pbix)
* CSV Dataset (.csv)
* Power Query
* DAX (Data Analysis Expressions)
* Data Modeling

---

## Project Files

* HR-Employee-Attrition.csv – Raw employee dataset
* HR Analytics Dashboard.pbix – Power BI dashboard file

---

## Dashboard Features

The dashboard includes:

* Total Employees
* Attrition Count
* Attrition Rate Percentage
* Active Employees
* Average Age
* Department-wise Attrition
* Job Role Analysis
* Salary and Attrition Comparison
* Gender Distribution
* Years at Company Analysis

Interactive filters allow users to explore data by department, job role, gender, education field, and other employee attributes.

---

## Key Insights

* Identifies departments with high attrition rates
* Analyzes job roles most affected by employee turnover
* Examines the relationship between salary and attrition
* Tracks employee demographics and workforce distribution
* Helps HR teams understand patterns influencing employee retention
